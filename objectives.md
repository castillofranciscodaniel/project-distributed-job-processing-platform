## Proyecto: Distributed Job Processing Platform (MVP profesional)

**Vamos a hacer:**
- API en Go
- Worker en Go
- SQS
- PostgreSQL (RDS o local + deploy simple)
- S3
- Docker
- Deploy en AWS (ECS o EC2 simple)
- Métricas + logs estructurados
- Retry + idempotencia real

*Nada más. Nada menos. Sin sobreingeniería.*
